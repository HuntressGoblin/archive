# The Full-Page Parchment

This page has a **full-page parchment theme** applied. Every part of the page — background, sidebars, TOC, headers, content — uses parchment colors and classic serif fonts.

## Subheading Example

Text and headings inside this page follow the parchment theme. The page background is soft beige, the sidebar is lighter beige, and headings are dark brown.

> [!note] This is a nested callout styled as a secondary surface. Notice the slightly darker background and brown border.

All elements like links, paragraphs, and nested callouts inherit the theme automatically.
